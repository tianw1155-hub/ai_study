### 今天讨论的内容

**最终确定的 MVP 网站（3个界面）：**

1. **首页**
   - 待确认内容（PRD评审等）
   - 执行进度（实时展示5人：你 + 4个agents 在干什么）
   - 工具调用展示（每个 agent 调用了哪些工具）

2. **任务看板**
   - 4个 agent 的任务状态（Kanban：待处理 → 进行中 → 完成 / 失败）
   - 不做房间动画（MVP 简化版）
   - 状态：待处理 / 进行中 / 等待中 / 已完成 / 失败

3. **产物交付**
   - 所有历史 PRD 版本（用户可查看任意版本）
   - 代码链接跳转 GitHub
   - 支持回退（PRD 回退 → 代码同步回退）
   - 展示：完成时间、总耗时、测试覆盖率
