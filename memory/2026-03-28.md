## 经验总结
1. `npx create-next-app` 交互式安装依赖太慢，subagent 无法等待。以后用**直接写文件**代替脚手架命令
2. JWT 认证 placeholder 代码要及时替换真实验证逻辑，不能留在生产路径上
3. 开发任务并行：前后端同时开工效率更高
4. 测试工程师可以与开发并行，提前发现问题
